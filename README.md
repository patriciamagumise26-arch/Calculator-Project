 Interactive Python Calculator

This is a simple, user-friendly command-line calculator built with Python. It allows users to perform basic arithmetic operations through an interactive loop.
 Features
 
Operations: Supports Addition, Subtraction, Multiplication, and Division.
Flexible Input: Recognizes both single letters (A, S, M, D) and full operation names (e.g., 'Addition').
Interactive Loop: Allows multiple calculations in one session without restarting the script.
Input Validation: Gracefully handles invalid choices to ensure a smooth user experience.

 How it works

The script uses standard Python functions for mathematical logic and a while True loop to keep the program running until the user chooses to exit.

 Usage


Run the script in a Python environment or Google Colab.
Choose an operation from the menu.
Enter the two numbers you wish to calculate.
Type 'yes' to perform another calculation or 'no' to exit.
